# Student Performance Analysis

## Project Overview

This project analyzes student academic performance using Python. The objective is to identify the factors that influence students' final grades and gain insights from the dataset through data analysis and visualization.

## Dataset Information

* Dataset: Student Performance Dataset
* Total Records: 395 Students
* Total Features: 33 Attributes
* Target Variable: G3 (Final Grade)

## Tools and Libraries Used

* Python
* Pandas
* Matplotlib
* VS Code
* Git & GitHub

## Data Cleaning

The dataset was examined for missing values and inconsistencies.

Steps performed:

* Loaded the dataset using Pandas.
* Checked dataset structure and data types.
* Verified missing values.
* Generated descriptive statistics.

## Exploratory Data Analysis (EDA)

The following analyses were performed:

### 1. Gender-wise Performance Analysis

Compared average final grades between male and female students.

### 2. Study Time vs Final Grade

Analyzed the relationship between study time and academic performance.

### 3. Failures vs Final Grade

Investigated the impact of previous academic failures on final grades.

### 4. Absences vs Final Grade

Studied how student absences affect academic performance.

### 5. Family Support Analysis

Examined whether family educational support influences student grades.

### 6. Internet Access Analysis

Compared student performance based on internet availability.

## Visualizations

The project includes:

* Bar Charts
* Histograms
* Scatter Plots

These visualizations help understand patterns and relationships within the data.

## Key Findings

* Students who spend more time studying generally achieve higher grades.
* Previous academic failures negatively affect final performance.
* Excessive absences tend to reduce academic achievement.
* Family support shows a positive influence on student outcomes.
* Earlier grades (G1 and G2) are strong indicators of final grade (G3).

## Conclusion

The analysis demonstrates that study habits, attendance, and academic history significantly impact student performance. By identifying these factors, educators and students can make informed decisions to improve learning outcomes.
