# Student Performance Analysis

## Objective
Analyze factors affecting student academic performance using Python.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis
- Study Time vs Grades
- Absences vs Grades
- Failures vs Grades

## Key Findings
- Higher study time generally leads to better grades.
- Previous failures negatively impact performance.
- Excessive absences can reduce academic results.

## Conclusion
Study habits and attendance are important factors influencing student performance.
